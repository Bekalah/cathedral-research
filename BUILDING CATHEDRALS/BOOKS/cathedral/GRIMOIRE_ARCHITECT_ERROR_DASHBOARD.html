Cathedral Error Dashboard

TypeScript Syntax in JS Files
  • Type annotations, property initializers, and signatures must only be used in .ts files.
  • Fix: Move TypeScript code to .ts or rewrite as valid JS.
Property/Type Errors in TypeScript
  • Type definitions must match actual object structure.
  • Fix: Update interfaces/types to match data.
Syntax Errors in HTML Embedded Scripts
  • Check for missing semicolons, brackets, and correct function/class structure.
General JS Syntax Issues
  • Misplaced brackets, missing semicolons, or incorrect function/class structure.
Error Prevention System:
  • Use ESLint and Prettier (auto-fix on save).
  • Enable VS Code extensions: auto-rename-tag, bracket-pair-colorizer, Prettier, code-spell-checker.
  • Enforce correct file extensions: .ts for TypeScript, .js for JavaScript.
  • Add type-checking and linting to CI.
  • Document best practices in the Grimoire Architect Build Map.
